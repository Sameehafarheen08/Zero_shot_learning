80 object classes recognized by the detection model

ℹ️ Why "pen" is not detected:
The current model is trained on the COCO (Common Objects in Context) dataset which contains 80 object classes. "Pen" is not included, so when you point at a pen, the model makes the best guess from similar objects like toothbrush, knife, or fork.
❌ Objects NOT Recognized by Current Model:
Pen, Pencil, Eraser, Marker, Highlighter, Crayon, Rubber band, Paperclip, Thumbtack, Scissors (as a tool), Stapler, Tape, Post-it note, and many other specific items.

💡 Solutions to Detect Pens & Other Custom Objects:

  1. Use a Larger YOLO Model: Switch from yolov8n (nano) to yolov8m (medium) or yolov8l (large) for slightly better accuracy, though pen still won't be recognized
  2. Switch to a Different Model: Use a model trained on office supplies or fine-grained object detection
  3. Fine-tune the Model: Collect images of pens and retrain YOLOv8 on custom data
  4. Use Image Classification Instead: Switch to CLIP for zero-shot classification - can identify any object described in text
  5. Combine Multiple Models: Use YOLOv8 for known objects + CLIP for custom objects
